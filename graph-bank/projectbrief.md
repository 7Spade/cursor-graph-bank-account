# Project Brief

## 專案資訊
- **名稱**：Angular Fire Rolekit
- **類型**：AI 輔助開發與權限系統樣板
- **版本**：0.0.0（開發階段）

## 技術規格
- **前端**：Angular 20.1.0 + Angular Material 20.1.0
- **後端**：Firebase 11.10.0 + @angular/fire 20.0.1
- **建置**：@angular/build + TypeScript 5.8.2
- **UI 主題**：Material Design azure-blue

## 功能範圍
- **權限管理系統**：完整的角色權限控制
- **組織管理**：多組織架構支援
- **用戶管理**：用戶註冊、認證、個人資料
- **儲存庫管理**：專案儲存庫管理功能
- **儀表板系統**：管理員、編輯者、檢視者介面

## 專案狀態
- **階段**：VAN 技術驗證完成
- **複雜度**：Level 3（中級功能）
- **下一步**：進入 PLAN 模式進行任務規劃
